cjc-admin-base
==============

Ansible role that tnstalls a few default packages I like having around when working on a Debian/Ubuntu system.

Packages
--------

 * *htop
 * *tmux
 * *vim-nox
 * *nmap
 * *nload
 * *curl
 * *nmap
 * *git
 * *iptraf
 * *strace
 * *mtr
 * *vnstat
 * *tcptrack
 * *socat
 * *apachetop
